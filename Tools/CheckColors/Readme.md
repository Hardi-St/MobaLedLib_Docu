Mit diesem Programm von Harold können die Helligkeiten und Farben der LEDs Live verstellt werden. Es ist ein kompiliertes Python Skript. Wegen seiner Größe und weil in einer Arduino Bibliothek keine EXE Programme enthalten sein dürfen ist das Programm hier abgelegt. 

Die Quelldateien zu dem Programm befinden sich in der Bibliothek. Der Benutzer kann entscheiden ob er sich einen Python Interpreter installiert oder die EXE Datei herunter lädt.

Das Programm wird automatisch vom Prog_Generator gestartet wenn man die „Set_ColTab“ Funktion benutzt. Es kann aber auch über das Optionsmenü aufgerufen werden.
