Momentan funktioniert das automatische herunterladen des ATTinyCore Pakets nicht weil das HTTPS Zertifikat der Webseite abgelaufen ist.
Als Work Around kannst die Angehängte ZIP Datei in das Verzeichnis
   "%USERPROFILE%\AppData\Local\Arduino15\packages"
entpackt werden

Das Paket brauchst man aber nur, wenn man den ATTiny programmieren will (Servo oder Charlieplexing Modul).
