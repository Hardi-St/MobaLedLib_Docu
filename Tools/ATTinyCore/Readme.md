Momentan funktioniert das automatische herunterladen des ATTinyCore Pakets nicht weil das HTTPS Zertifikat der Webseite abgelaufen ist.
Als Work Around kann die angehängte ZIP Datei in das Verzeichnis
   "%USERPROFILE%\AppData\Local\Arduino15\packages"
entpackt werden

Das Paket braucht man aber nur, wenn man den ATTiny programmieren will (Servo oder Charlieplexing Modul).
